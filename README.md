# Probabalistic-Robotics
This folder contains the code and supplementary report for the coursework assignment for the Probabilistic Robotics module for BEng Robotics.  
  
Particularly of note are the implementations of an Extendend Kalman Filter and a Particle Filter, written in MATLAB, to localise a simulated robot moving through a maze, which comprised tasks 3 & 4 respectively.  
  
Assignment mark: 97%
